## Business Problem
A UK-based mid-sized e-commerce retailer wants to understand sales performance and customer purchasing behaviour to support data-driven decisions around revenue growth, product strategy, and operations.

## Objectives
- Analyse sales trends over time
- Identify top-performing products and categories
- Compare regional performance
- Create an executive-level performance dashboard

## Tools Used
- SQL
- Power BI
- Microsoft Excel
- Python

## Key Deliverables
- Cleaned and structured dataset
- SQL queries answering core business questions
- Interactive Power BI dashboard
- Business insights and recommendations
